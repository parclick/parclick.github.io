Perfil ideal

Por encima de tus capacidades técnicas, valoramos tu actitud, calidad humana y trabajo en equipo, un equipo técnico cohesionado es nuestro mayor valor.


Es necesario que cumplas los siguientes requisitos:
  • Amplia experiencia con PHP y Symfony
  • Experiencia en desarrollo y consumo de APIs. Estar familiarizado con todo lo que conlleva: arquitectura, cache, monitorización, versionado, testeo, etc.
  • La calidad importa. Estar familiarizado con SOLID y con buenas prácticas de diseño y código.
  • Toda nuestra API está testada con PHPUnit, por lo que será una herramienta que usarás a diario
  • Estar acostumbrado a trabajar con revisión de código y pair programming
  • Dominio de GIT
  • Experiencia trabajando y diseñando bases de datos relacionales, utilizamos MYSQL
  • Experiencia analizando y diseñando aplicaciones
  • Mentalidad ágil. Iterar rápido sin sacrificar la calidad para entregar el valor al usuario final lo antes posible.

Además, valoraremos positivamente si tienes:
  • Experiencia usando diferentes servicios de AWS
  • Experiencia trabajando con Docker
  • Conocimientos de Golang/Python
  • Experiencia con sistema de mensajería RabbitMQ, SQS, SNS, etc.
